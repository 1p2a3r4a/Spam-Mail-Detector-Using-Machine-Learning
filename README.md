# Spam Mail Detector Using Machine Learning

## 📌 Project Overview

This project is a Machine Learning and Natural Language Processing (NLP) application that classifies SMS messages as **Spam** or **Ham (Not Spam)**.

The project demonstrates the complete machine learning workflow, including:

* Data Loading
* Exploratory Data Analysis (EDA)
* Text Preprocessing
* Feature Extraction using TF-IDF
* Model Training with Naive Bayes
* Model Evaluation
* Spam Prediction on New Messages

---

## 📂 Dataset

**Dataset:** SMS Spam Collection Dataset

The dataset contains SMS messages labeled as:

* **Spam** – Unwanted promotional or fraudulent messages.
* **Ham** – Normal, legitimate messages.

## 📊 Project Workflow

1. Import required libraries
2. Load the dataset
3. Explore and understand the dataset
4. Clean and preprocess text
5. Convert labels into numerical values
6. Transform text using TF-IDF Vectorizer
7. Split the dataset into training and testing sets
8. Train the Naive Bayes classifier
9. Evaluate model performance
10. Predict whether new SMS messages are Spam or Ham
